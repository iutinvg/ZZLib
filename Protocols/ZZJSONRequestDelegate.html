ZZJSONRequestDelegate Protocol Reference

Conforms to NSObject

Overview

Delegate to handle request async loading.

It must be used for controller which uses ZZJSONRequest.

Parameters

Discussion

Invocated when loading is failed with error.

This method allows to show error alerts, disable loading visualizing.

Warning: You must call super class method to release request.

Discussion

Default implementation of [ZZJSONRequestDelegate requestDidFinishLoading:]

Must be overriden in order to get loaded data. The super class method must be invoked at the end of overriden method. Example:

NSDictionary tmp = (NSDictionary)request.response; [searchResults release]; _searchResults = [[NSArray alloc] initWithArray:[tmp objectForKey:@“results”]];

// reload table [self.tableView reloadData];

// in case you use ZZTableViewCell [self loadVisibleImages];

// must be called to release request // and hide loading visualizations [super requestDidFinishLoading:request];
