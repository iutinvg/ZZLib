ZZViewController Class Reference

Inherits from UIViewController
Conforms to ZZJSONRequestDelegate
Declared in ZZViewController.h

Overview

Short hand for creation of view controller with white view stretched to full screen and with support of all interface orientations.

Please see Demo part of ZZLib project as a best example of the usage:

  • https://github.com/iutinvg/ZZLib/blob/master/Demo/ZZDemoViewController.h
  • https://github.com/iutinvg/ZZLib/blob/master/Demo/ZZDemoViewController.m

showLoading:

Creates/removes a view for visualizing of loading process.

- (void)showLoading:(BOOL)flag

Parameters

flag

YES to creates loading view, NO to remove

Discussion

Creates/removes a view for visualizing of loading process.

You will need it in case of usage this class in pair with ZZJSONRequest or similar. Default implementation is very simple: white background with activity indicator in the center. Please override if necessary. It is not necessary to involve the super class method in your one, though you may of course.

Usage example:

- (void)viewDidLoad {
    [super viewDidLoad];
    // starting loading process here
    [self showLoading:YES];
}
