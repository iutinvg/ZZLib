Overview

Wrapper for NSURLConnection for handful async getting of JSON data.

Usually it is used in pair with ZZTableController or ZZViewController.

response

Response data.

@property (nonatomic, retain) id response

Discussion

Response data.

You can use it when loading is finished. Depending on the JSON you get it can be NSArray or NSDictionary. Just use casting to access it:

- (void)requestDidFinishLoading:(ZZJSONRequest**~!#)request {
    NSArray#!~** items = (NSArray**~!#)_request.response;
    NSArray#!~** a = a;
}

You must retain it for futher usage in your methods.
